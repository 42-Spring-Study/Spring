## Study Rules 🚨
### Online
- 매주 정해진 양의 강의 수강 및 정리
- 정리 내용은 섹션 단위로 게시 (`강의명/섹션명/이름.md`)
- 질문 사항은 이슈 생성 (`[강의명/섹션명]질문 제목`)
- 해결된 이슈는 닫기
### Offline
- 매주 수요일 저녁 7시 모임
- 주에 배운 내용에 대해 발표 진행
- 발표자와 발표 파트는 랜덤으로 선정
- 해결되지 않은 이슈에 대한 이야기 나눔
- 다음 주 스터디 진도 정하기
